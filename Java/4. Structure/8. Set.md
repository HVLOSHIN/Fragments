### [[5. List|List]] vs Set
###### List
- **순서** : 리스트에 추가된 요소는 순서를 유지한다. 추가된 순서를 반영할 수 있다.
- **중복** : 동일한 값이나 객체의 중복을 허용한다.
- **인덱스** : 리스트의 각 요소는 인덱스를 통해 접근 가능

###### Set
유일한 요소들의 컬렉션
- **유일성** : 중복을 허용하지 않는다. 이미 존재하는 요소를 추가하면 무시된다.
- **순서 미보장** : 요소들의 순서를 보장하지 않는다.
- **빠른 검색** : 요소의 유무를 빠르게 찾을 수 있도록 최적화 되어있다.
